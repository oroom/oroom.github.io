Yoga.com classes

Yoga app for yoga.com. Has a lot of features

  • Inspire: a global social Yoga and Fitness Community. Universal customizable module used also in sport.com apps.
  • Yoga poses and breathing exercises with advanced search and filtering
  • Video demonstrations for all poses
  • Predefined and custom (user created) yoga programs with video instructions
  • Social integrations, deep analytics and A/B testing
  • Functional in offline (all content can be pre downloaded, social module has local db cache for viewed content)

One of the best yoga apps on the market with more than 7M installs for paid version.

Check on itunes

Environment

Objective-C, RestKit, Cocoapods

Role

Implementing first version of native social module with local data storage “FitnShare” (something like instagram for fitness). Module was fully customizable using css-like syntax files for simple integration in the different apps. This module was released as separate app later - https://itunes.apple.com/US/app/id977991432 (not by me) Working on yoga app updates, ios7 redesign with UI based on serverside configs for A/B testing support. Working on other common modules for all apps.
